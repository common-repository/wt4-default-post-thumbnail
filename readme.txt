Display a default thumbnail for posts and post types in WordPress

== Description ==

Display a default thumbnail for posts and post types in WordPress, use the the_post_thumbnail feature

== Installation ==

To setup this in your theme, make sure it uses the_post_thumbnail to call the thumbnail and then add an image by:

e.g.

1. go to appearance > customize > site identity
2. Upload an image in the "post thumbnail" section
